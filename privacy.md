## What we collect and why

This site does not run analytics, does not use tracking pixels, and has no user accounts or contact forms. The personal data processed in connection with visiting this site comes from two infrastructure layers:

### GitHub Pages (hosting)

This site is hosted on [GitHub Pages](https://pages.github.com/) (GitHub, Inc. / Microsoft Corporation). GitHub may collect server log data including IP addresses, browser user-agent strings, referrer URLs, and timestamps as part of normal service operation. See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) for details. We do not have access to these logs.

### Cloudflare (CDN and security)

Traffic is proxied through [Cloudflare, Inc.](https://www.cloudflare.com/) for performance and DDoS protection. Cloudflare processes connection metadata (IP address, request headers) and may set strictly-necessary security cookies (see [Cookie Policy](/cookies/)). See [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/) for details.

**Legal basis:** legitimate interest in operating a secure and functional website (Art. 6(1)(f) GDPR).

## Data retention

We do not retain personal data ourselves. GitHub and Cloudflare retain data according to their own policies.

## Your rights

Under the GDPR you have the right to access, rectify, erase, restrict, or port your personal data, and to object to processing based on legitimate interest. To exercise these rights, contact [info@lab42.it](mailto:info@lab42.it). You also have the right to lodge a complaint with the Italian supervisory authority: [Garante per la protezione dei dati personali](https://www.garanteprivacy.it).
